Project details

Northstar Analytics dashboard redesign.

A responsive dashboard built to help operations managers understand daily performance without digging through scattered reports.

Overview

The original dashboard had useful data, but the layout made every number feel equally important. I reorganized the experience around priority signals, clearer metric cards, and a cleaner responsive grid.

Project facts

  • Type: SaaS dashboard
  • Role: UX structure, visual design, front-end build
  • Tools: HTML, CSS Grid, JavaScript for menu only
  • Result: faster scanning and clearer reporting flow

Challenge

The team needed to see risk, progress, and performance in one calm view. The previous design forced users to compare too many disconnected blocks.

Design decisions

I used white cards, subtle borders, blue accents for priority values, and consistent headings so managers could compare data quickly.

Outcome

The final concept reduced visual noise and gave the team a dashboard structure they could expand with future reports.
