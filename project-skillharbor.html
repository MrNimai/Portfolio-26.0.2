Project details

SkillHarbor learning platform concept.

A learning interface that helps students browse courses, understand progress, and return to the right lesson quickly.

Overview

SkillHarbor was designed around learner confidence. Instead of showing endless course tiles, the interface highlights progress, next lessons, categories, and practical course outcomes.

Project facts

  • Type: Learning platform UI
  • Role: UX layout and responsive interface
  • Tools: HTML, CSS, accessible cards
  • Result: easier browsing and clearer learning paths

Challenge

Students often leave learning platforms when they cannot quickly decide what to continue or start next.

Design decisions

I separated recommended lessons, progress cards, categories, and course benefits so every section had a clear job.

Outcome

The concept feels approachable, organized, and useful for both new learners and returning students.
