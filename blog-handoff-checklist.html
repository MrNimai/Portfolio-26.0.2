A calmer handoff checklist.

A small checklist can prevent many common problems before a static website is shared or published.

Before handing off a site, check the things that affect real users first: links, forms, mobile layout, focus states, and content accuracy.

Check every link

Navigation links, card links, buttons, and footer links should go where the visitor expects. Placeholder `#` links should be replaced or removed.

Review content and images

Every image should have useful alt text. Every section should have real writing, not temporary filler.

Test the small screen

Resize the browser or open the site on a phone. Make sure text does not overlap, buttons are readable, and cards stack cleanly.
